# Samarth Shinde — Portfolio

A modern, animated personal portfolio built as a single-file HTML/CSS/JS application — no frameworks, no dependencies, just pure web craft.

## Live Demo

> https://samarthfolio.netlify.app

## Features

- **Neural Network Canvas** — 100 interactive particles that react to mouse movement
- **Loading Screen** — animated percentage counter on entry
- **Custom Cursor** — morphing ring + dot cursor with hover effects
- **Typing Animation** — cycles through roles (AI Analyst, ML Researcher, etc.)
- **3D Tilt Cards** — project cards respond to mouse with perspective transforms and glare
- **Animated Skill Bars** — progress bars fill on scroll with glowing tips
- **Draw-in Timeline** — experience timeline line draws itself on scroll
- **SVG Progress Rings** — CGPA displayed as animated circular indicators
- **Number Counters** — stats animate from 0 on scroll-in
- **Scroll Progress Bar** — thin gradient bar at top of page
- **Fully Responsive** — works on mobile, tablet, and desktop

## Tech Stack

- Pure **HTML5 / CSS3 / Vanilla JavaScript**
- Google Fonts: Syne · Inter · JetBrains Mono
- Canvas API for particle network
- IntersectionObserver API for scroll animations
- No external libraries or frameworks
